Рядом с исполняемым файлом "MoskvinWorkers.exe" должна быть папка "Settings".

Внутри этой папки также есть папки. Их описание ниже.
1) Папка "LastSession". Здесь находится файл "LastSession.xml", в котором хранится список файлов, которые пользователь использовал при последнем запуске программы.
2) Папка "Favorits". Здесь находится файл "Favorits.xml", в котором хранится список файлов, которые пользователь добавил в Избранное. 
    Это делается для тех случаев, когда пользователь еще раз добавляет файл в список синхронизаций, то автоматически будут загружаться те пути, которые были добавлены в избранное.
3) Папка "Program" содержит файл "ProgramSettings.xml", в котором хранятся все настройки приложения.
